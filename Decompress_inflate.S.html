val eval : t -> [ `Error | `Flush of int | `Ok of int ]
eval inflater performs inflater. The value os eval inflater is:
  • `Ok complete the document
  • `Flush n wait to flush the dst for re-writing a news values in dst; n bytes were read from the source
  • `Error a very bad state
